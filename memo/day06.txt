1. 반복문

1) do ~ while 문
	while 문과 동일. 한번은 무조건 실행해야 할 때 사용한다.

2) 이중 for문
	하나의 반복문 안에 또 다른 반복문을 포함하는 구조

	for(초기식 ; 조건식 ; 증감식){ 		// 바깥 for문
		for(초기식; 조건식; 증감식){		// 안쪽 for문
		}
	}

3) 기타 제어문
	반복문에서 사용한다.
	단, break는 switch에서도 사용한다.

	3.1 break
		즉시 해당 중괄호 영역을 탈출
		나머지 코드는 실행하지 않는다.

	3.2 continue
		즉시 다음 반복으로 넘어간다.
		아래 코드를 실행시키지 않는다.
